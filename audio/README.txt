Taruh file musik backsound kamu di sini dengan nama: song.mp3

Contoh: audio/song.mp3

index.html sudah mereferensikan file ini lewat:
<audio id="bg-music" src="audio/song.mp3" loop></audio>

Kalau mau pakai nama/format lain (misalnya song.ogg atau lagu.mp3),
tinggal ubah atribut src di index.html sesuai nama filenya.
